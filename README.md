# Chinook Music Store SQL Analysis

## Project Overview
This project analyzes the Chinook Music Store database using SQL to uncover business insights related to customer behavior, sales performance, music preferences, and revenue trends.

## Dataset
- Database: Chinook Music Store
- Tables Used: Customers, Invoices, Invoice_Items, Tracks, Albums, Artists, Genres, Employees

## Tools Used
- SQL
- MySQL
- GitHub

## Key Business Questions Solved
- Top selling genres
- Best performing artists
- Highest revenue generating countries
- Customer spending analysis
- Monthly sales trends
- Repeat customer analysis
- Employee sales performance

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOINS
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions (RANK, DENSE_RANK)

## Files Included
- Chinook SQL sheet.sql

- Project Presentation.pdf
